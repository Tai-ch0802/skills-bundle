# Anthropic 品牌樣式

## 概覽

要存取 Anthropic 的官方品牌識別和風格資源，使用此技能。

**關鍵字**：品牌、企業識別、視覺識別、後製處理、樣式、品牌色彩、排版、Anthropic 品牌、視覺格式化、視覺設計

## 品牌指南

### 色彩

**主要色彩：**

- 深色：`#141413` - 主要文字與深色背景
- 淺色：`#faf9f5` - 淺色背景與深色上的文字
- 中灰：`#b0aea5` - 次要元素
- 淺灰：`#e8e6dc` - 微妙背景

**強調色：**

- 橘色：`#d97757` - 主要強調
- 藍色：`#6a9bcc` - 次要強調
- 綠色：`#788c5d` - 第三強調

### 排版

- **標題**：Poppins（以 Arial 作為備用）
- **內文**：Lora（以 Georgia 作為備用）
- **注意**：字體應預先安裝於你的環境中以獲得最佳效果

## 功能

### 智慧字體套用

- 將 Poppins 字體套用至標題（24pt 及以上）
- 將 Lora 字體套用至內文
- 自訂字體無法使用時，自動回退至 Arial/Georgia
- 在所有系統上保持可讀性

### 文字樣式

- 標題（24pt+）：Poppins 字體
- 內文：Lora 字體
- 根據背景智慧選擇色彩
- 保留文字層級和格式

### 形狀與強調色

- 非文字形狀使用強調色
- 循環使用橘色、藍色和綠色強調
- 在保持品牌一致性的同時維持視覺趣味

## 技術細節

### 字體管理

- 可用時使用系統安裝的 Poppins 和 Lora 字體
- 提供自動回退至 Arial（標題）和 Georgia（內文）
- 無需安裝字體 — 可與現有系統字體搭配使用
- 為獲得最佳效果，預先在你的環境中安裝 Poppins 和 Lora 字體

### 色彩套用

- 使用 RGB 色值進行精確的品牌色匹配
- 透過 python-pptx 的 RGBColor 類別套用
- 在不同系統間維持色彩保真度
